Stewart es un RPG de fantasía ligera, aventura y comedia que combina estética furry, aventura medieval y cultura colombiana. Inspirado en Deltarune, hecho en Godot y siempre en busca de nuevos participantes. 🦝💖

El jugador acompaña a **Stewart**, un alce autoproclamado caballero, junto a sus amigos **Zarah** y **Gabi**, en un viaje épico y ridículo a través de siete reinos llenos de misterios, humor y trasfondo oscuro. Inspirado en *Deltarune*, *Hollow Knight* y *The Valiant Prince Ivandoe*, entre otros.

- **Público objetivo:** Adolescentes y adultos (+13).
- **Plataformas:** Escritorio
- **Estado actual:** Prototipo en desarrollo (este repo).

---

## Requerimientos y dependencias
- Godot 4.6.1 estable (opcionalmente la versión .NET)
- Git y GitHub
- Pixieditor y algún programa de dibujo
- Visual Studio Code u otro editor de tu gusto

## Contribución (si participas en Stewart DevTeam)
Puedes usar **GitHub Desktop** para gestionar todo esto de forma más sencilla.

1. Asegúrate de tener la última versión de la rama principal:
    ```bash
    git checkout main
    git pull
    ```

2. Crea una rama para tu tarea:
    ```bash
    git checkout -b <tipo>/<nombre-descriptivo>
    ```

3. Realiza y commita tus cambios
4. Sube los cambios a tu rama:
    ```bash
    git push origin <nombre-de-la-rama>
    ```

5. Abre un Pull Request para revisión.
